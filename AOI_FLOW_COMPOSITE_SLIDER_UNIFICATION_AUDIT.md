# AOI FLOW 合成タブ 調整バー統一 監査メモ

## 目的
合成タブ内で背景・商品・影の調整バーが二重に表示されていたため、操作UIを1系統に統一する。

## 修正内容
- `ProductPlacementEditor.tsx` 内の重複していた調整バー群を削除。
- 背景ズーム・背景左右・背景上下・商品サイズ・商品位置・影調整は1セットだけ表示。
- 「座標固定」ボタンは維持。
- 背景 → 座標固定 → 商品 → 影 → 合成 の流れは維持。
- 上部プレビュー集約方針は維持。

## 変更ファイル
- `app/flow/drafts/new/components/ProductPlacementEditor.tsx`
